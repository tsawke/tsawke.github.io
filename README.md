# Yuxuan Hou — Personal Research Portfolio

Source for [tsawke.github.io](https://tsawke.github.io/), a lightweight static research portfolio built with semantic HTML, modern CSS, and a small amount of vanilla JavaScript.

## Local preview

From the repository root, run any static server, for example:

```powershell
python -m http.server 8000
```

Then open `http://127.0.0.1:8000/`.

The site has no build step or runtime dependencies and is ready for GitHub Pages deployment from the repository root.
